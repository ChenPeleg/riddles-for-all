# Script Recipes

This document provides a collection of script recipes that can be used for various tasks. Each recipe includes a brief
description and the corresponding script code.

# Guidelines for Writing Scripts

1. use esm syntax (import/export)
2. use async/await for asynchronous operations
3. include error handling
4. each script should have a class. the class should have a run method that executes the main logic of the script.
5. the script should be in the `job` files folder.
